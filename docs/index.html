L* Algorithm Visualization Example

Target Language: Strings over {a, b} containing an even number of a's

Step 1: Initial Observation Table

Starting with empty string (ε) in S, and single letter suffixes in E

ε a
S: ε 1 0
S·A: a 0 1
S·A: b 1 0

1 = accept, 0 = reject

Step 2: Observation Table After Adding 'aa'

Table is not closed because row(a) is not in S. Adding 'aa' to S.

ε a
S: ε 1 0
S: aa 1 0
S·A: a 0 1
S·A: b 1 0
S·A: aaa 0 1
S·A: aab 1 0

Step 3: First Conjecture DFA

States:

  • q0 = [row(ε)] - Initial and accepting state
  • q1 = [row(a)] - Non-accepting state

Transitions:

  • q0 --a--> q1
  • q0 --b--> q0
  • q1 --a--> q0
  • q1 --b--> q1

This DFA correctly accepts strings with an even number of a's!

Key Observations:

  • The table starts with S = {ε} and E = {ε, a}
  • We fill the table using membership queries (is the string in the language?)
  • The algorithm detected the table wasn't closed (row(a) wasn't in S)
  • After adding 'aa', we got a consistent and closed table
  • The resulting DFA has 2 states and correctly recognizes strings with even number of a's
